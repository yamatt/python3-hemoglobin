# Hemoglobin

Hemoglobin is a CLI wrapper for GrammarBot so that it can be used to check your content for errors.

## Installation

### Pip

```bash
pip install hemoglobin
```

### UV

```bash
uv install hemoglobin
```

## Running Locally

You will need [uv](https://docs.astral.sh/uv/getting-started/installation/).

```bash
uv run -m pip install hemoglobin
```

## Usage

```bash
uv run -m hemoglobin exampleApiKey ./content
```
